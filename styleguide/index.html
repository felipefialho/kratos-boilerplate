Atoms

Buttons

Button styles can be applied to any element. Typically you'll want to use either a <button> or an <a> element:

<button class="btn">Click</button>
<a class="btn" href="/some-page">Some Page</a>

Forms

The <form> element represents a document section that contains interactive controls to submit information.

Input

The HTML element <input< is used to create interactive controls for web-based forms in order to accept data from the user.

<input class="form-control" placeholder="Input example">
Select

The <select> element represents a control that presents a menu of options.

<select class="form-control">
  <option>Select example</option>
</select>
Label

The <label> represents a caption for an item in a user interface.

<label>Label example</label>
Legend

The <legend> represents a caption for the content of its parent.

<legend>Legend example</legend>

Typography

Links

The <a> element specifies relationships between the current document and an external resource.

<p>Lorem ipsum dolor sit amet, <a class="#">consectetur adipisicing elit</a>. Optio, beatae.</p>
Headings

The heading elements implement six levels of document headings, <h1> is the most important and <h6> is the least. A heading element briefly describes the topic of the section it introduces.

<div class="content">
  <h1>Lorem ipsum</h1>
  <h2>Lorem ipsum</h2>
  <h3>Lorem ipsum</h3>
  <h4>Lorem ipsum</h4>
  <h5>Lorem ipsum</h5>
  <h6>Lorem ipsum</h6>
</div>
Paragraph

The <p> element represents a paragraph of text.

<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Inventore omnis consequatur eligendi.</p>
Blockquote

The <blockquote> element indicates that the enclosed text is an extended quotation.

<blockquote>
  <cite>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Voluptatum nesciunt laborum.</cite>
  <footer><p>Lorem ipsum dolor sit</p></footer>
</blockquote>
